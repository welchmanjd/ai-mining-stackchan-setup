AIマイニングスタックチャン セットアップ
================================

このアプリは、AIマイニングスタックチャン（M5Stack Core2）に
ファームウェアを書き込み、Wi-FiやAPIキーなどの設定を投入するための
セットアップツールです。

使い方（概要）
------------
1. USBでM5Stack Core2をPCに接続します。
2. AiStackchanSetup.exe を起動します。
3. 画面の手順に従ってファーム書き込みと設定を行います。
4. 完了後、AIマイニングスタックチャンが起動します。

実行ファイル
------------
このフォルダ内の `AiStackchanSetup.exe` を実行してください。
他のファイルは必須の部品なので、同じフォルダに置いたままにしてください。

ログ
----
ログは次のフォルダに保存されます：
  %LOCALAPPDATA%\AiStackchanSetup\Logs

トラブル時
----------
「サポート用ログを作成」ボタンでZIPを作成し、
そのZIPをサポート窓口に提供してください。
